설치법

3단계면 끝. 1분이면 설치 완료.

1

ZIP 다운로드

아래 다운로드 섹션에서 비밀번호를 입력하고 ZIP 파일을 받으세요.

2

압축 해제 + 복사

압축을 풀고 ppt-slide-generator 폴더를 skills 폴더에 넣으세요.

3

Claude Code 재시작

터미널에서 Claude Code를 다시 열면 자동 인식됩니다. "PPT 만들어줘" 입력!

# macOS / Linux (Downloads 폴더에서 압축 풀었다면)
cp -r ~/Downloads/ppt-slide-generator ~/.claude/skills/

# Windows (PowerShell)
Copy-Item -Recurse "$env:USERPROFILE\Downloads\ppt-slide-generator" "$env:USERPROFILE\.claude\skills\"

# 다른 경로에 압축 풀었다면 해당 경로로 바꿔주세요

나만의 테마 만들기

직접 만들 필요 없이, Claude Code한테 시키면 됩니다

2단계면 끝

Step 1. 폴더 준비

로고 파일과 브랜드 가이드라인(있다면)을 한 폴더에 넣어주세요. PDF, 이미지, 뭐든 OK.

Step 2. Claude Code에 말하기

"내 브랜드 테마 만들어줘. 로고랑 가이드라인은 [폴더 경로]에 있어" 라고 입력하면 자동으로 테마를 생성합니다.

# Claude Code에서 이렇게 말하세요
"내 브랜드로 PPT 테마 만들어줘. 브랜드 자료는 C:/Users/me/brand/ 에 있어"
폴더에 뭘 넣으면 되나요?

로고 파일(PNG/SVG)만 있어도 됩니다. 브랜드 가이드라인 PDF, 컬러 팔레트 이미지, PPT 템플릿 등이 있으면 더 정확한 테마가 만들어집니다. Claude Code가 파일을 분석해서 컬러, 폰트, 디자인 언어를 자동으로 추출합니다.
